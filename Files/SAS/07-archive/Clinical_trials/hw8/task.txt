Remove laboratory tests, that:

a) contain ONLY missing results;
b) DO NOT have any numeric results;
After that remove results with +-<> symbols. For those tests and results, that remain, consider any character result as missing. Count Mean, Median, SD, Min, Max and number of missing results for each lab test. Create dataset with SUBJID, TESTNAME, (Mean, Median, SD, Min, Max, NMiss - num, no label) variables. Sort by SUBJID, TESTNAME.

Count frequencies for all responses (1-5) for a) overall; b) within each treatment group. Convert responses to character (1=Bad; 2=Poor; 3=Satisfactory; 4=Good; 5=Excellent).
Use ANY preparation steps and any procedures you like. Create dataset with following structure: ARM (standard label, $50 - full TRT Group name as in RDMCODE dataset), ARMCD (standard label, $1 - TRT Group code), QSTEST (standard label, $100 - with values "Number of '<response>' Answers" where <response> is one of {Bad; Poor; Satisfactory; Good; Excellent}), QSSTRESN (standard label, num - holds the result).
Add one extra record for ARM/ARMCD = "Overall"/"O". Sort by ARMCD, QSTEST.

VISIT (standard label, $50, just set to VISID);
