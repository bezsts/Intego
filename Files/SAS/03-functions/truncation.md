# Округлення та усічення

## Швидка довідка

| Функція | Що робить | Приклад |
|---------|-----------|---------|
| `round(x, mult)` | До найближчого кратного | `round(1.25, 0.1)` → `1.3` |
| `ceil(x)` | Найменше ціле ≥ x | `ceil(2.1)` → `3`, `ceil(-2.4)` → `-2` |
| `floor(x)` | Найбільше ціле ≤ x | `floor(2.9)` → `2`, `floor(-2.4)` → `-3` |
| `int(x)` | Ціла частина (до нуля) | `int(2.9)` → `2`, `int(-2.9)` → `-2` |

## Рецепти

### Типові округлення в SDTM/ADaM
```sas
/* Вік (завжди вниз — floor) */
age = floor(yrdif(birthdt, visitdt, "ACTUAL"));

/* Значення VS (до 0.1) */
vsstresn = round(raw_value * 2.54, 0.1);     /* дюйми → см */
vsstresn = round(raw_value * 0.45359, 0.1);  /* фунти → кг */
vsstresn = round((raw_f - 32) / 1.8, 0.1);  /* F → C */

/* CHG та PCHG */
chg  = round(aval - base, 0.01);
pchg = round((aval - base) / base * 100, 0.01);
```

### Різниця між FLOOR та INT для від'ємних чисел
```sas
/* INT рухається до нуля */
int(-2.9)   /* → -2 */
floor(-2.9) /* → -3 */

/* Для позитивних — однаково */
int(2.9)    /* → 2 */
floor(2.9)  /* → 2 */
```

### ROUND із різною точністю
```sas
round(1234.567, 100)    /* → 1200 */
round(1234.567, 10)     /* → 1230 */
round(1234.567, 1)      /* → 1235 */
round(1234.567, 0.1)    /* → 1234.6 */
round(1234.567, 0.01)   /* → 1234.57 */
```
