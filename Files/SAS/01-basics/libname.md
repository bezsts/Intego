# Libname — підключення бібліотеки

## Що робить
Дає SAS "псевдонім" для папки на диску. Після цього можна звертатися до датасетів як `lib.dataset`.

## Синтаксис
```sas
libname <ім'я> '<шлях>';
```
- Ім'я ≤ 8 символів, починається з літери або `_`.

## Рецепти

### Підключити бібліотеку
```sas
libname raw "C:\Data\Rawdata";
libname out "C:\Data\Output";

data out.result;
    set raw.source;
run;
```

### Тимчасова бібліотека (WORK)
```sas
/* WORK — автоматично доступна, видаляється після сесії */
data work.temp;   /* те саме що просто: data temp; */
    set raw.source;
run;
```

### Перевірити, що бібліотека підключена
```sas
proc contents data=raw._all_ nods; run;
```

## Правила іменування
| ✅ Правильно | ❌ Неправильно |
|---|---|
| `raw` | `rawdata_folder` (> 8 символів) |
| `out1` | `1out` (починається з цифри) |
| `_tmp` | `my-lib` (дефіс заборонений) |
