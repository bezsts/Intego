# Special-Purpose Domains

Об'єднує домени, які мають унікальну структуру і не підпадають під загальну логіку "спостережень"

## DM

Зберігає інформацію про характеристики суб'єкта (вік, стать, раса) та ключові дати участі в дослідженні (згода, початок/кінець лікування).

**<font color="red">1 record per subject</font>**

| Змінна | Опис | Додаткова інформація |
| --- | --- | --- |
| SUBJID | Унікальний номер суб'єкта в межах лікарні | |
| RFSTDTC | Початок дослідження | Зазвичай це дата першої дози або рандомізації, рідше — підписання Informed Consent. |
| RFENDTC | Закінчення участі | Дата останньої дози, візиту або вимірювання (assessment). |
| RFXSTDTC | Дата початку лікування | |
| RFXENDTC | Дата закінчення лікування | |
| RFICDTC | Дата підписання Informed Consent | |
| RFPENDTC | Остання відома дата активності суб'єкта в базі даних | |
| DTHDTC | Дата та час смерті суб'єкта | |
| DTHFL | "Прапорець" смерті | Вказує на факт смерті суб'єкта. Дорівнює або "Y", або NULL |
| SITEID | Номер центру/лікарні | |
| INVID | Код дослідника | |
| INVNAME | Ім'я дослідника | |
| BRTHDTC | Дата народження | |
| AGE | Вік | Зазвичай різниця між RFSTDTC та BRTHDTC. |
| AGEU | Одиниці виміру | |
| SEX | Стать | |
| RACE | Раса | |
| ETHNIC | Етнос | |
| ARM | Заплановане лікування | |
| ARMCD | Скорочена версія | Макс. 20 символів |
| ACTARM | Фактичне лікування | Може відрізнятися від запланованого |
| ACTARMCD | Скорочена версія | Можливі значення:<br><br>UNPLAN — незаплановане лікування.<br><br>NOTTRT — рандомізований, але не отримав ліки.<br><br>SCRNFAIL — провалив скринінг.<br><br>NOTASSGN — пройшов скринінг, але не рандомізований. |
| COUNTRY | Країна лікарні | Використовується codelist C66786 |
